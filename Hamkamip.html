🚀 التثبيت والإعداد

متطلبات النظام

  • Node.js 16.0 أو أحدث
  • npm 8.0 أو أحدث
  • RAM 4GB كحد أدنى
  • مساحة تخزين 2GB

خطوات التثبيت

1. تحميل المشروع

git clone https://github.com/3RBAI/wolf-ai.git

2. الانتقال للمجلد

cd wolf-ai

3. تثبيت المكتبات

npm install

4. تشغيل المشروع

npm start

إعداد متغيرات البيئة

إنشاء ملف .env

WOLF_AI_API_KEY=your_api_key_here
WOLF_AI_ENDPOINT=https://api.wolfai.com
WOLF_AI_MODEL=wolf-v2
WOLF_AI_LANGUAGE=ar
WOLF_AI_REGION=mena

💡 نصائح التثبيت

  • • تأكد من وجود اتصال إنترنت مستقر أثناء التثبيت
  • • استخدم Node.js LTS للحصول على أفضل استقرار
  • • قم بإعداد API Key من لوحة التحكم أولاً
  • • اختبر الاتصال بـ API قبل البدء في التطوير

📱 دليل الاستخدام

💬 واجهة المحادثة

تفاعل مباشر مع الذكاء الاصطناعي من خلال واجهة محادثة بديهية:

  • • اكتب سؤالك في صندوق النص
  • • استخدم الأيقونات لإرفاق الملفات
  • • احفظ المحادثات المهمة
  • • شارك الردود مع فريقك

🎨 إنشاء المحتوى

أنشئ محتوى متنوع بالذكاء الاصطناعي:

  • • كتابة المقالات والمدونات
  • • إنشاء منشورات وسائل التواصل
  • • تصميم الشعارات والهويات
  • • ترجمة النصوص

مثال على الاستخدام الأساسي

import WolfAI from 'wolf-ai-sdk';

const wolf = new WolfAI(process.env.WOLF_AI_API_KEY);

// إرسال سؤال باللغة العربية
const response = await wolf.chat({
  message: "اكتب لي مقال عن الذكاء الاصطناعي",
  language: "ar",
  context: "business"
});

console.log(response.content);

🔌 وثائق API

نقاط النهاية الرئيسية

POST /api/v1/chat محادثة

إرسال رسالة والحصول على رد من الذكاء الاصطناعي

POST /api/v1/generate إنشاء

إنشاء محتوى مخصص حسب المتطلبات

GET /api/v1/models معلومات

الحصول على قائمة النماذج المتاحة

مثال على طلب API

curl -X POST https://api.wolfai.com/v1/chat \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "مرحبا",
    "language": "ar",
    "max_tokens": 1000
  }'

مثال على الاستجابة

{
  "status": "success",
  "data": {
    "message": "مرحبا! كيف يمكنني مساعدتك اليوم؟",
    "tokens_used": 15,
    "model": "wolf-v2",
    "language": "ar"
  }
}
